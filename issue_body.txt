Hva du må gjøre (sikkert):
1) Klon repo på nytt:

   git clone https://github.com/Einars-Apps/Matsjekk.git

Alternativ (avansert, kan overskrive lokale endringer):

   git fetch origin --prune
   git checkout main
   git reset --hard origin/main

Advarsel: Jeg brukte force-push på remote; lokale kloner som inneholder commits fra gammel historie kan få konflikter eller miste commits hvis du ikke tar forholdsregler. Kontakt meg hvis du trenger hjelp med å redde lokal arbeid.

Takk  gi beskjed her i issuet hvis noe går galt eller du trenger hjelp.
