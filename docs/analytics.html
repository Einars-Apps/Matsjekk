Analytics

Oversikt

Denne siden fungerer som dokumentasjon og hjelpe-side for Google Analytics / verifisering.

1) For å aktivere sporing (GA4)

  1. Gå til Google Analytics og finn din Measurement ID (format: G-XXXXXXXXXX).
  2. Åpne docs/analytics.html eller docs/index.html og lim inn gtag-snippet i <head>, eller fjern kommentarene rundt gtag-blokken ovenfor og erstatt MEASUREMENT_ID med din ID.
  3. Commit og push endringene til repo, og la GitHub Pages publisere siden.

2) Search Console / site verification

Hvis du trenger å verifisere eierskap i Search Console via meta-tag, legg meta-taggen i docs/index.html <head> og push. Eksempel:

<meta name="google-site-verification" content="GV5-nGUT1Sq6DlinAh5DvlnnfyzaKJ5Q7DyjQ38X868" />

3) Sikkerhet og personvern

Husk at du må oppdatere Personvern-siden for å beskrive hva som spores og hvorfor. Ads and analytics must be disclosed and consented to via the cookie banner.
